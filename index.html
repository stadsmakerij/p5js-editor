Basis

Elk programma heeft twee blokken: setup() draait 1x zodra je start, draw() herhaalt zichzelf 60 keer per seconde.

Het canvas is je tekenblad — een leeg vel waar alles op verschijnt wat je tekent. Je maakt het aan in setup().

Waar iets komt bepaal je met twee getallen: x is hoe ver van links, y is hoe ver van boven. Hoe groter het getal, hoe verder naar rechts (x) of naar beneden (y).

Dus op een canvas van 400×400:

• linksboven: (0, 0)
• rechtsboven: (400, 0)
• linksonder: (0, 400)
• rechtsonder: (400, 400)
• midden: (200, 200)

createCanvas(breedte, hoogte) createCanvas(400, 400); // maak een tekenblad van 400x400

Vormen

Let op: bij circle en ellipse is (x, y) het midden. Bij square en rect is (x, y) de linkerbovenhoek. Bij triangle geef je drie aparte punten.

circle(x, y, grootte) circle(200, 200, 50); // cirkel met midden op (200, 200), grootte 50 ellipse(x, y, breedte, hoogte) ellipse(200, 200, 80, 50); // ovaal met midden op (200, 200), 80 breed, 50 hoog square(x, y, grootte) square(100, 100, 50); // vierkant met linkerbovenhoek op (100, 100), grootte 50 rect(x, y, breedte, hoogte) rect(100, 100, 80, 60); // rechthoek linksboven (100, 100), 80 breed, 60 hoog triangle(x1, y1, x2, y2, x3, y3) triangle(200, 100, 150, 200, 250, 200); // driehoek tussen drie punten line(x1, y1, x2, y2) line(0, 0, 100, 100); // lijn van (0, 0) naar (100, 100) point(x, y) point(50, 50); // puntje op (50, 50)
Wat je later tekent, ligt bovenop wat je eerder tekende. Zet background() dus meestal als eerste in draw().

Kleuren

Een kleur maak je met drie getallen: rood, groen, blauw. Elk getal gaat van 0 (niets) tot 255 (vol).

fill(rood, groen, blauw) fill(255, 0, 0); // vulkleur rood fill(0, 255, 0); // vulkleur groen fill(0, 0, 255); // vulkleur blauw fill(255, 255, 0); // geel (rood + groen) fill(255, 255, 255); // wit (alles vol) fill(128, 128, 128); // grijs (alle drie gelijk) fill(0, 0, 0); // zwart (alles 0) background(rood, groen, blauw) background(220, 220, 220); // grijze achtergrond noFill(); // geen vulling stroke(rood, groen, blauw) stroke(0, 0, 255); // lijnkleur blauw strokeWeight(dikte) strokeWeight(5); // lijndikte 5 pixels noStroke(); // geen rand/lijn

Tekst

text(tekst, x, y) text("Hallo!", 100, 100); // tekst op positie (100, 100) textSize(grootte) textSize(32); // tekstgrootte

Slimme trucjes

Venster-afmetingen

Met windowWidth en windowHeight krijg je de afmetingen van het browservenster. Handig om een canvas te maken dat het hele scherm vult.

createCanvas(breedte, hoogte) createCanvas(windowWidth, windowHeight); // canvas zo groot als het venster

Canvas-afmetingen

In plaats van vaste getallen kun je ook width (breedte van je canvas) en height (hoogte) gebruiken. Handig als je iets altijd in het midden of op de rand wil hebben.

circle(x, y, grootte) circle(width/2, height/2, 50); // cirkel precies in het midden circle(width, height, 50); // cirkel rechtsonder rect(x, y, breedte, hoogte) rect(0, 0, width, height); // rechthoek over het hele canvas

Muispositie

Met mouseX en mouseY weet je waar de muis staat. Zet je tekening in draw() zodat het elk frame meebeweegt.

circle(mouseX, mouseY, 50); // cirkel volgt de muis line(0, 0, mouseX, mouseY); // lijn van linksboven naar de muis

Willekeurig

Met random() krijg je elke keer een ander getal. Handig voor verrassingen in je tekening.

random(max) random(100); // willekeurig getal tussen 0 en 100 random(min, max) random(50, 150); // willekeurig tussen 50 en 150 random() random(); // willekeurig getal tussen 0 en 1 random(lijst) random(["appel", "peer", "banaan"]); // willekeurig item uit een lijst
